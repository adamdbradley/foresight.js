Dynamic Image URI Template Examples

This demo uses images which can be dynamically resized by requesting specific dimensions directly within the image request. Common examples includes a directory with dimension values, such as /images/640x480/larry.jpg, querystring values, such as /images/curley.png?w=640&h=480, or filenames prefixed with the width dimensions, such as /images/640px-moe.gif.

Foresightjs.appspot.com has a few methods available to dynamically server-resize images according to the dimensions within the src URI. Each of the examples below use foresight's various dynamic image resizing just as an example of the different formats possible.
