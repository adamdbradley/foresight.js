Static Image URI Replacement Examples

This demo replaces existing scale factors and browser width/height within static image request URIs. If foresight.js does not see a change from the browser width/height to the request width/height it will not do any dimension replacing. If the browser width/height is different than the request width/height than it will find and replace those dimensions within the img src. View the source CSS to see how it works.
